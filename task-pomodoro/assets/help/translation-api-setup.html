翻译配置教程

默认翻译使用本地按需索引查词，不联网、不写剪贴板、不保存翻译历史。只有你主动配置并启用 API 后，选中文本才会发送到对应服务。

自动取词依赖目标软件暴露 UIA 选中文本；如果读不到选区，可启用剪贴板监听后手动复制英文文本作为兜底。程序不会模拟 Ctrl+C，也不会临时写入或恢复剪贴板。

API 配置

自定义接口

适合你已有自己的翻译服务或代理。程序会向你填写的地址发送 JSON：

{ "text": "hello", "source": "auto", "target": "zh" }

接口返回 JSON 即可：

{ "translation": "你好" }

如需鉴权，把密钥填入 API key。程序会用 Windows 当前用户 DPAPI 加密保存，但你仍应自行保存原始密钥。

DeepL

  1. 打开 DeepL Translate API 文档。
  2. 确认你的账号类型：Free 使用 https://api-free.deepl.com，Pro 使用 https://api.deepl.com。
  3. 在程序里选择 DeepL，选择 Free 或 Pro，填入 API key，然后点击“测试”。

百度翻译

  1. 打开 百度翻译开放平台通用翻译 API 文档。
  2. 创建应用后复制 App ID 和密钥。
  3. 在程序里选择百度翻译，填入 App ID 和 Secret，然后点击“测试”。

在线 API 可能产生服务商费用或额度消耗。本程序默认月度上限为 100000 字符，可在翻译设置中调整。

完整版词典

默认安装包内置核心词典和同名索引，保证程序轻、启动快、离线可用。完整版词典作为独立文件发布，文件格式应保持为本系统可直接导入的 TSV，并建议随附同名 .idx 索引：

word    phonetic    pos    translation    tags    frequency    exchange

推荐文件名：task-pomodoro-full-dictionary.tsv。下载后解压为 TSV，再在翻译设置里点击“导入词典”。

  • GitHub 发布页：minimal-desktop-pomodoro releases
  • Gitee 镜像：镜像仓库和 release asset 稳定后再在此处补充。

点击“获取完整版”时，程序会按当前策略尝试远程源；失败后检查本机缓存 task-pomodoro/data/dictionaries/task-pomodoro-full-dictionary.tsv 和开发目录 local-assets/dictionaries/task-pomodoro-full-dictionary.tsv。首次成功后后续默认本地优先。已绑定词典时，“导入词典”会变为“解绑词典”。

词典来源和许可需随文件说明一起发布；当前默认词典由 ECDICT 生成，遵循 MIT license。
